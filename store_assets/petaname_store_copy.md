## 詳しい説明

【顔と名前、覚えるのが苦手なあなたへ】
「名前を覚えよう」は、顔と名前を楽しく覚えるパーティ＆記憶トレーニングゲーム。遊んでいるうちに、自然と「名前を覚える力」がきたえられます。

▼ メインモード「なまえコール」
① 出てきたキャラに、すきな名前をつける
② カードがもう一度出てきたら、その名前を思い出して答える
③ 早く正しく答えた人がカードをゲット！集めた枚数で勝負

・👤 ひとりで：自分のペースで名前を覚える脳トレに
・🎉 みんなで（1台）：スマホを囲んで2〜4人、名前を叫んで早い者勝ち！
・🌐 オンライン：ほかのプレイヤーと名前あてレース

命名スタイルは「まとめて命名」と「出たとき命名」から選べて、遊ぶたびに新鮮。

▼ 📸 自分の写真で覚える「おぼえる」モード
職場や学校で出会う人の顔写真と名前を登録して、フラッシュカードで学習＆クイズ。新しい人の名前を覚えたいときにぴったりです。

▼ もうひとつの遊び方「ペアさがし」
顔カードと名前カードをそろえる神経衰弱。CPU対戦で段位を上げたり、記憶術のコツを学べる読み物も。

▼ こんな人におすすめ
・人の名前がなかなか覚えられない
・接客・営業・イベントで名前を覚える機会が多い
・スキマ時間に頭を使うゲームを探している
・家族や友だちとワイワイ遊べるパーティゲームがほしい

さあ、遊びながら「名前を覚える力」をきたえよう！

## English — Short description

Bad with names? Train your face-and-name memory with a fun matching game!

## English — Full description

PetaName is a memory-training game built for one specific struggle:
"I remember the face, but the name just won't come out."

HOW IT WORKS
1. Memorize: study each person's face and name 👀
2. The cards flip over and shuffle!
3. Match each face card to its name card 🏷️

Pairings are randomized every game — no rote memorization,
just real encode-and-recall practice each time you play.

LEARN REAL MNEMONICS
Techniques said to be used by memory athletes, taught in-app:
tagging first impressions, turning names into images, and the memory palace.
Guided Mnemonic Training mode walks you through them as you play.

MODES
- 🧠 Solo Training: scored by moves and time, 3 difficulty levels
- 🤖 CPU Battles: take turns flipping — most pairs wins, ratings rise
- 📊 Training Report: accuracy, decision speed, and streak feedback
- 🏆 Achievements, titles, themes, and cheering pups

Note: This app is not a medical device or treatment program. It does not
guarantee improvement of memory or cognitive function; results vary.
